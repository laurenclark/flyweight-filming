== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-customize-login-page` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings/wp-customize-login-page Options to edit your settings.
4. Set your customizations.
5. checkbox check for "enable" in status box.
